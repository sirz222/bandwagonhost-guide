## 打不开怎么办？按顺序试这几步

搬瓦工新域名虽然是备用的，但有时候也会遇到访问不稳定的情况。遇到打不开，按这个顺序来：

1. **换一个镜像域名**。bwh81.net 打不开，试 bwh88.net，再试 bwh89.net。这几个通常不会同时出问题。
2. **刷新本地 DNS 缓存**。Windows 打开命令提示符，输入 `ipconfig /flushdns`，回车。Mac 和 Linux 执行 `sudo dscacheutil -flushcache`（macOS）或 `sudo systemd-resolve --flush-caches`（部分 Linux 发行版）。
3. **换公共 DNS**。把设备的 DNS 服务器改成 8.8.8.8（Google）或 1.1.1.1（Cloudflare），对解决 DNS 污染通常有效。
4. **修改 hosts 文件**（适合有一定技术基础的用户）。在 hosts 文件里添加：
   
   104.31.86.112 bwh81.net
   104.31.86.112 www.bwh81.net
   
   这样可以直接绕过 DNS 解析，强制指向正确的 IP。
5. **提工单联系客服**。如果以上都不行，登录 KiwiVM 面板或能访问的任何镜像域名，提交工单，内容写 "Can you tell me your newest mirror website"，客服一般几个小时内回复，会给你最新可用的域名。
